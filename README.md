## Description

Using Node.js, Inquirer, and MySQL to create an application to manage a company's employee database.

## Installation

```
git git@github.com:zahrahaji10/employee-tracker-system.git
cd employee-tracker-system
npm install
```

## Launch Server

```
nmp run start
nmp run dev
```
